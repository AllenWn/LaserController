# DRV2605 (Haptic Driver)

Source: `design_src/datasheet/drv2605.pdf`

## Interface
- I2C control interface.
- Supports single-byte and multi-byte register R/W, including sequential addressing.

## I2C Address
- 7-bit device address: `0x5A` (binary `1011010`)
- 8-bit address bytes: write `0xB4`, read `0xB5`

## Key Pins (per datasheet pin list)
- `EN`: device enable input.
- `IN/TRIG`: multi-mode input (I2C selectable as PWM, analog, or trigger).
- `SDA`, `SCL`: I2C.
- `VDD`, `GND`
- `OUT+`, `OUT-`: motor drive outputs.

## Firmware-Relevant Notes For This PCB
- `EN` and `IN/TRIG` are each routed to ESP32 GPIOs.
- `SCL/SDA` are routed to ESP32 GPIOs (separate I2C bus vs shared bus depends on your pin mux choices).
