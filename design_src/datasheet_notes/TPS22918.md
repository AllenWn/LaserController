# TPS22918 (Load Switch)

Source: `design_src/datasheet/tps22918.pdf`

## Function
- N-channel load switch with controlled output slew rate via `CT`.
- Optional quick output discharge via `QOD`.

## Supply / Inputs (high level)
- VIN operating range noted: `1.0V to 5.5V`.
- Device operates regardless of power sequencing order.
  - If voltage applied to `ON` before `VIN`, the slew rate of `VOUT` is not controlled (per datasheet note).

## Pins (per datasheet pinout)
- Pin 1: `VIN`
- Pin 2: `GND`
- Pin 3: `ON`
- Pin 4: `CT` (slew rate control; can be left floating)
- Pin 5: `QOD` (quick output discharge control)
- Pin 6: `VOUT`

## CT Slew Control
- Datasheet provides an approximate relationship between `CT` and `VOUT` slew rate and includes recommended CT values for inrush limiting.

## Firmware-Relevant Notes For This PCB
- `ON` is controlled by an ESP32 GPIO (`GN_LD_EN` net on the PCB).
